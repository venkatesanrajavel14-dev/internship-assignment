Observations

  • Block elements (`div`): Always start on a new line and take up the full width available.
  • Inline elements (`span`): Do not start on a new line; they only take up as much width as their content.
  • Block elements are useful for creating sections or containers, while inline elements are used for styling parts of text within a line.
